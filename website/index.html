Quick Installation

Since the extension isn't on the Chrome Web Store yet, it takes 3 quick steps to install.

1

Download and Unzip

Click the download button above to get the `.zip` archive. Extract the folder to a safe location on your computer.

2

Enable Developer Mode

Open Chrome and navigate to chrome://extensions in your URL bar. Toggle the Developer mode switch in the top right corner.

3

Load the Extension

Click the Load unpacked button that just appeared in the top left, and select the unzipped extension folder you previously downloaded.
